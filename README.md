# Mood Music Recommender
I made a simple Python project using Tkinter. My program recommends songs based on the mood you have inputed

# What it does:
-You can choose mood from drop down menu (like happy, sad, chill, etc.)
-Program randomly picks song depending on the choosen mood
-You can click a button to open the recommened song on Spotify

# What I used for project:
-Python
-Tkinter (which is to make app window and buttons)
-random (to pick songs randomly)
-webbrowser (so I can open Spotify links)

# Why I made this:
I made because I enjoy listening to music and it constantly changes depending on how I feel, and I wanted to learn how to make simple apps with buttons so I combined them both and made something that I can use as well.
